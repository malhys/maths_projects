\begin{myex}
	Au lancer d'un dé cubique, on considère les événements $A$ : <<Obtenir au moins 5>>, et $B$ : <<Obtenir un nombre pair>>. On a :
	\begin{multicols}{2}


	\begin{itemize}
		\item $A = \{5 ; 6\}$.
		\item $B = \{2 ; 4 ; 6\}$.
		\item $\bar{A} = \{1 ; 2 ; 3 ; 4\}$.
		\item $A \cap B = \{6\}$.
		\item $A \cup B = \{2 ; 4 ; 5 ; 6\}$.
	\end{itemize}
	
	\end{multicols}
\end{myex}
